GrovePi
=======

GrovePi is an open source platform for connecting Grove Sensors to the Raspberry Pi.

Started 2013/10/04
See more at the [GrovePi Site](http://dexterindustries.com/GrovePi/)
[Dexter Industries](http://www.dexterindustries.com)
